### NPTEL Vinternship – IIT Ropar

## Description

This Case Study demonstrates MVC Pattern in Express. It demonstrates how to building a Scalable System with MVC and Modular Design in Express.

## Steps to Run

1. **Extract** the zip file and navigate to the project directory:

    ```
    cd WIN251113_Express_CS_8
    ```

2. **Install** dependencies:

    ```
    npm install
    ```

3. **Run** the application:
    ```
    ts-node src/app.js
    ```
4. **Open** in brower by Ctrl + Click on Localhost URL
