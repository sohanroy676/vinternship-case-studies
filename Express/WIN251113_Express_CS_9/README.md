## Description

This Case Study demonstrates Repository Pattern in Express. It demonstrates how to create repository interfaces to abstract data access, implement repositories for different storage types (memory, file, database) and swap storage backends without changing business logic.

## Steps to Run

1. **Extract** the zip file and navigate to the project directory:

    ```
    cd WIN251113_Express_CS_9
    ```

2. **Install** dependencies:

    ```
    npm install
    ```

3. **Run** the application:
    ```
    ts-node src/app.js
    ```
4. **Open** in brower by Ctrl + Click on Localhost URL
